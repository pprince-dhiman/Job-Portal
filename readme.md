# -> Application Flow <-

1. Recruiter posts a job,
2. Student applies for the job,
3. Application stored in Application collection,
4. Recruiter reviews applicants,
5. Recruiter updates application status,
6. Student get response to applied job in real time.
 |-> Accepted or Rejected.

 
#  ->Tech Stack<-
# Frontend

React.js, Redux Toolkit, Tailwind CSS, Axios.

# Backend

Node.js, Express.js, MongoDB (Mongoose), 
JWT Authentication, Cookie-based auth
